How to use UOB Internet Banking on Chrome/Safari on MAC OS X Lion

Jan 08, 2012 in Project

I have been suffering with login issues on UOB internet banking site for as long as I can remember. I had to use Windows on a virtual machine to always login on UOB internet banking site.

I checked java installation and its all in top shape. I knew that its some crappy javascript coding that is causing it to fail to load (or merely reporting it as “failed to load”). Surprisingly site got loaded perfectly on Firefox and Opera on OS X Lion. Puh! After searching internet for possible issue I ran into a chromium issue report 40364. That made me wear my developer hat.

I started looking up for the error message all over the javascript and html code. I started by stepping through code in the “Developer Tools” in chrome. Main culprit here was initApplet() method in pib_login.js.

All you need to do is execute ‘initApplet’ and ‘init’ method on the login page after everything is finished loading. I tried writing an extension to DO THAT , but Chrome’s extension protects the page javascript and css. It allows you to modify DOM only. So, I had to think of something else. And then I remembered “Bookmarklet”. And here it is now. Just follow the steps below to get it working.

  1. Drag the link below (“FIX UOB Login”) to your bookmark bar (one time only) or create a bookmark with address/URL as one mentioned below (Updated).
  2. Goto UOB internet banking page.
  3. On the login page, after you get the error (see image) click “OK”, let the page load fully
  4. Once page is fully loaded, click on the bookmarklet (that you dragged in first step).
  5. That’s it you can login now.
  6. Repeat step 2-5, every time you want to login.
